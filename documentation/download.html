Linux/Shell Installer

Clone the repository:

git clone https://github.com/Nolann-Epi/my_rpg.git 
git clone https://github.com/ulysse-qd/my_rpg.git 
git clone https://github.com/Arcahub/my_rpg.git
git clone https://github.com/ThomasMarches/my_rpg.git

Download the version of your distrib on this link

Then install it with:

sudo yum localinstall mysql80-community-release-{distrib version}-{version-number}.noarch.rpm

Install CSFML

sudo ./csfml/build_csfml.sh

To compil:

make

Update :

git pull && make re

How to play

./my_rpg -h

Launch the game

./my_rpg
